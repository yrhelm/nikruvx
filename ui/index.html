      Model Gate — security regression suite for any LLM

      Model spec format: ollama:llama3.1, openai:gpt-4o-mini, anthropic:claude-3-5-sonnet-20241022, copilot:gpt-4o, stub:demo (offline demo). API keys are read from OPENAI_API_KEY / ANTHROPIC_API_KEY / COPILOT_TOKEN.
      Pick a model spec and click Run. The default stub:demo uses a built-in refusal-style stub so you can see the suite end-to-end without any API keys.
      Regression mode: compare a candidate model against a baseline. Surfaces only the probes that newly fail in the candidate (and, separately, probes the candidate fixes).
      Multi-turn auditor-vs-target audits from the Petri framework (Anthropic / Meridian Labs). One LLM tries to coerce another across 5-12 turns to surface failure modes single-turn probes cannot reach: gradual jailbreaks, system-prompt extraction, deception under pressure, eval awareness, sycophancy, value drift, multi-turn PHI extraction, chained prompt injection.
      Pick a scenario and Run. Each audit takes 30-90 seconds with real models. The default stub:* specs run instantly so you can verify the UI without API keys.
