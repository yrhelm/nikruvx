# NikruvX docs/

This folder holds long-form documentation, reference artifacts, and assets
that don't belong in the runtime tree.

## Contents

| Path | Purpose |
|---|---|
| `openapi.json` | Snapshot of the FastAPI OpenAPI spec, regenerated by `make openapi` / `.\tasks.ps1 openapi` |
| `screenshots/` | UI screenshots referenced from the README and Wiki |

## Live API docs

When the server is running you can hit:

| URL | What it is |
|---|---|
| http://127.0.0.1:8000/docs | Swagger UI - interactive try-it-now |
| http://127.0.0.1:8000/redoc | ReDoc rendering of the same spec |
| http://127.0.0.1:8000/openapi.json | Raw OpenAPI 3.x spec |

The static `docs/openapi.json` here is the version pinned at the last
release - regenerate after any endpoint changes:

```bash
make openapi          # Linux / macOS
.\tasks.ps1 openapi   # Windows
```

The CI pipeline also produces `docs/openapi.json` as a build artifact on
every push to `main`.
